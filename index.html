Development

Quotes on Dev was a solo school project that uses the WordPress REST API to retrieve a random post at the user’s request as well as submit a new post from the user. I learned how to use and manipulate the API to deliver the desired information.

Vesta is a custom built WordPress theme that was started as a school project and has continued as a freelance project. As the Technical Lead for the group portion of the project, I was responsible for merging my team members' code, maintaining functionality and resolving merge conflicts. Our team worked effectively in a Scrum environment to complete the desktop version on schedule. The mobile site is currently in progress. Please check back later when the site goes live!

InhabiTENT is a WordPress theme built from scratch for a fictional camping gear company. This solo project familiarized me with PHP, WordPress’s built-in functions and its hierarchy. It also provided me with an opportunity to hone my styling skills using Sass.

Instanews is a responsive one-page website that allows the user to choose their desired news category. It then displays that categories' top 12 stories from the New York Times. Throughout the project I learned how to use APIs and jQuery’s ajax method as well as implementing a CSS grid.

Pong is a game built entirely in JavaScript that allows users to play an old-style pong game by moving paddles up and down to deflect a ball and score on the opposing player. This project provided excellent practice in implementing and organizing JavaScript modules and SVGs.

Aloha is a responsive, one-page website for a fictional apparel company. This solo project helped me refine my responsive design skills in HTML and CSS.
